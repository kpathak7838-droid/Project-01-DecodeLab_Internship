# DecodeLabs_Internship
Project Overview

This project was developed as part of the DecodeLabs Full Stack Development Internship Program. The objective of Project 01 is to build a responsive and interactive frontend application using modern web development technologies.

The project demonstrates the fundamentals of frontend development, including component-based architecture, user interface design, and responsive layouts.

Objectives
Develop a modern frontend application.
Understand React component structure.
Create reusable UI components.
Implement responsive web design.
Practice frontend development workflow.

Technologies Used
HTML
CSS
JavaScript
